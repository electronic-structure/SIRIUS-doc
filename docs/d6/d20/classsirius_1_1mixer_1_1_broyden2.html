Detailed Description

template<typename... FUNCS>
class sirius::mixer::Broyden2< FUNCS >

Broyden mixer.

Second version of the Broyden mixer, which doesn't require inversion of the Jacobian matrix. Reference paper: "Robust acceleration of self consistent field calculations for density functional theory", Baarman K, Eirola T, Havu V., J Chem Phys. 134, 134109 (2011)

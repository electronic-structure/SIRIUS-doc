Inner product between wave-functions.

This function computes the inner product using a moving window scheme plus allreduce. The input wave-functions data must be previously allocated on the GPU. The result is always returned in the CPU pointer. In case of a single MPI rank the result is also returned in the GPU pointer.

The following \( m \times n \) sub-matrix is computed:

\[ S_{irow0+i,jcol0+j} = \beta S_{irow0+i,jcol0+j} + \langle \phi_{i0 + i} | \tilde \phi_{j0 + j} \rangle \]

Parameters
[in]bra"bra" wave-functions \( \phi \).
[in]i0index of the first "bra" wave-function.
[in]mnumber of "bra" wave-functions.
[in]ket"ket" wave-functions \( \tilde \phi \).
[in]j0index of the first "ket" wave-function.
[in]nnumber of "ket" wave-functions.
[in]beta\( \beta \) parameter.
[in,out]resultinner product matrix \( S \).
[in]irow0first row (in the global matrix) of the inner product sub-matrix.
[in]jcol0first column (in the global matix) of the inner product sub-matrix.

Linear transformation of the wave-functions.

The transformation matrix is expected in the CPU memory. The following operation is performed:

\[ \psi^{out}_{j} = \alpha \sum_{i} \psi^{in}_{i} Z_{ij} + \beta \psi^{out}_{j} \]

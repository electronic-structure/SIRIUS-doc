Linear transformation of the wave-functions.

The transformation matrix is expected in the CPU memory. The following operation is performed:

\[ \psi^{out}_{j} = \alpha \sum_{i} \psi^{in}_{i} Z_{ij} + \beta \psi^{out}_{j} \]

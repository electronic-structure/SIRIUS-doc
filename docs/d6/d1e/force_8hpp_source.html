80  /** In the second-variational approach we need to compute the following expression for the k-dependent
81  * contribution to the forces:
82  * \f[
83  * {\bf F}_{\rm IBS}^{\alpha}=\sum_{\bf k}w_{\bf k}\sum_{l\sigma}n_{l{\bf k}}
84  * \sum_{ij}c_{\sigma i}^{l{\bf k}*}c_{\sigma j}^{l{\bf k}}
85  * {\bf F}_{ij}^{\alpha{\bf k}}
86  * \f]
87  * This function sums over band and spin indices to get the "density matrix":
88  * \f[

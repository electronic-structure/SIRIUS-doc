Inner product between wave-functions.

This function computes the inner product using a moving window scheme plus allreduce. The input wave-functions data must be previously allocated on the GPU. The result is always returned in the CPU pointer. In case of a single MPI rank the result is also returned in the GPU pointer.

The following \( m \times n \) sub-matrix is computed:

\[ S_{irow0+i,jcol0+j} = \langle \phi_{i0 + i} | \tilde \phi_{j0 + j} \rangle \]

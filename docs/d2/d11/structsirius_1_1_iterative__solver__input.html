◆ converge_by_energy_

int sirius::Iterative_solver_input::converge_by_energy_ {1}

Defines the flavour of the iterative solver.

If converge_by_energy is set to 0, then the residuals are estimated by their norm. If converge_by_energy is set to 1 then the residuals are estimated by the eigen-energy difference. This allows to estimate the unconverged residuals and only then compute only the unconverged.

◆ orthogonalize_

bool sirius::Iterative_solver_input::orthogonalize_ {true}

Control the subspace expansion.

If true, keep basis orthogonal and solve standard eigen-value problem. If false, add preconditioned residuals as they are and solve generalized eigen-value problem.

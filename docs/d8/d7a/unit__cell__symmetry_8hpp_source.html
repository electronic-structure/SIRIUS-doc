241 /** \page sym Symmetry
242  * \section section1 Definition of symmetry operation
243  *
244  * SIRIUS uses Spglib to find the spacial symmetry operations. Spglib defines symmetry operation in fractional
245  * coordinates:
246  * \f[
247  * {\bf x'} = \{ {\bf R} | {\bf t} \} {\bf x} \equiv {\bf R}{\bf x} + {\bf t}
248  * \f]
249  * where \b R is the proper or improper rotation matrix with elements equal to -1,0,1 and determinant of 1
250  * (pure rotation) or -1 (rotoreflection) and \b t is the fractional translation, associated with the symmetry
251  * operation. The inverse of the symmetry operation is:
252  * \f[
253  * {\bf x} = \{ {\bf R} | {\bf t} \}^{-1} {\bf x'} = {\bf R}^{-1} ({\bf x'} - {\bf t}) =
254  * {\bf R}^{-1} {\bf x'} - {\bf R}^{-1} {\bf t}
255  * \f]
256  *
257  * We will always use an \a active transformation (transformation of vectors or functions) and never a passive
258  * transformation (transformation of coordinate system). However one should remember definition of the function
259  * transformation:
260  * \f[
261  * \hat {\bf P} f({\bf r}) \equiv f(\hat {\bf P}^{-1} {\bf r})
262  * \f]
263  *
264  * It is straightforward to get the rotation matrix in Cartesian coordinates. We know how the vector in Cartesian
265  * coordinates is obtained from the vector in fractional coordinates:
266  * \f[
267  * {\bf v} = {\bf L} {\bf x}
268  * \f]
269  * where \b L is the 3x3 matrix which clomuns are three lattice vectors. The backward transformation is simply
270  * \f[
271  * {\bf x} = {\bf L}^{-1} {\bf v}
272  * \f]
273  * Now we write rotation operation in fractional coordinates and apply the backward transformation to Cartesian
274  * coordinates:
275  * \f[
276  * {\bf x'} = {\bf R}{\bf x} \rightarrow {\bf L}^{-1} {\bf v'} = {\bf R} {\bf L}^{-1} {\bf v}
277  * \f]
278  * from which we derive the rotation operation in Cartesian coordinates:
279  * \f[
280  * {\bf v'} = {\bf L} {\bf R} {\bf L}^{-1} {\bf v}
281  * \f]
282  */

Detailed Description

Descriptor for the atomic radial functions.

The radial functions \( f_{\ell \nu}(r) \) are labeled by two indices: orbital quantum number \( \ell \) and an order \( \nu \) for a given \( \ell \). Radial functions can be any of augmented waves of local orbitals (in case of FP-LAPW) or bete projectors, atomic or Hubbard wave functions in case of PP-PW.

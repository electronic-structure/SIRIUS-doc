Detailed Description

template<typename T, typename U = double>
class sirius::Spline< T, U >

Cubic spline with a not-a-knot boundary conditions.

The following convention for spline coefficients is used: between points \( x_i \) and \( x_{i+1} \) the value of the spline is equal to \( a_i + b_i(x_{i+1} - x_i) + c_i(x_{i+1}-x_i)^2 + d_i(x_{i+1}-x_i)^3 \).

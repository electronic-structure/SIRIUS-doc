41  /// Retrieve or set the Hubbard occupancies.
42  /** This functions helps retrieving or setting up the hubbard occupancy
43  * tensors from an external tensor. Retrieving it is done by specifying
44  * "get" in the first argument of the method while setting it is done
45  * with the parameter set up to "set". The second parameter is the
46  * output pointer and the last parameter is the leading dimension of the
47  * tensor.
48  *
49  * The returned result has the same layout than SIRIUS layout, * i.e.,
50  * the harmonic orbitals are stored from m_z = -l..l. The occupancy
51  * matrix can also be accessed through the method occupation_matrix()
52  *
53  * \param [in] what String to set to "set" for initializing sirius ccupancy tensor and "get" for retrieving it.
54  * \param [inout] occ Pointer to external occupancy tensor.
55  * \param [in] ld Leading dimension of the outside tensor.
56  * \return return the occupancy matrix if the first parameter is set to "get". */
